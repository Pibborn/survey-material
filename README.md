1. Installation:

```bash
python -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

2. Usage:

```bash
python screening.py wos_again.csv
```

3. Important notes

* You can use the `--out` flag to save this into a custom .csv file
* You can use the `--theme solarized` flag to get some nicer colors
